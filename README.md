# Nuxt 3 Minimal Starter

## this is an example app using :
--> Javascript TS
--> vue.js version 3
--> nuxt.js version 3
--> express.jd
--> mongodb
--> tailwindcss


## RUN THE APP:
    1-  Install docker
    2-  execute : docker build -t todos-app-nuxt3 .
    3-  execute : docker run --name todosApp -p 5000:5000 todos-app-nuxt3
